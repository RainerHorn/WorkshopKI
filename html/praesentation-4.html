Beispiel A – Textzusammenfassung

Gruppenaufgabe: Einen längeren Text strukturiert zusammenfassen

📋 Aufgabe Kopiere einen Absatz aus einem Fachtext oder einem Schulbuch in den Chat. Bitte das Modell, eine Zusammenfassung in 3 Sätzen zu erstellen – für Schüler*innen der Klasse 10.
✅ Erwartetes Ergebnis Eine verständliche, altersgerechte Zusammenfassung. Vergleiche: Wie ändert sich das Ergebnis, wenn du die Zielgruppe oder Satzanzahl änderst?
💡 Tipp: Die Prompt-Formulierung steuert den Output entscheidend. Probiere: „Fasse in 3 Stichpunkten zusammen" vs. „Schreibe einen zusammenhängenden Absatz".

Beispiel B – Q&A auf eigene Inhalte

Gruppenaufgabe: Das Modell als Lernbegleiter nutzen

📋 Aufgabe Füge einen eigenen Text (z. B. Unterrichtsnotizen, Lernzettel) in den Prompt ein. Stelle dann gezielte Fragen dazu – lass das Modell nur auf Basis dieses Textes antworten.
✅ Erwartetes Ergebnis Antworten, die sich auf den bereitgestellten Text beziehen. Das Modell sollte bei Wissenslücken im Text zugeben, es nicht zu wissen.
🔍 Diskussion: Wann beantwortet das Modell Fragen aus dem Kontext – und wann erfindet es Antworten? Wie erkennt man Halluzinationen?

Beispiel C – Code & Automatisierung

Gruppenaufgabe: KI als Coding-Assistent

📋 Aufgabe (wähle eine):
  • Lass ein einfaches Shell-Skript oder Python-Skript schreiben
  • Bitte das Modell, einen vorhandenen Code zu erklären
  • Lass einen SQL-Ausdruck oder Regex-Muster generieren
✅ Erwartetes Ergebnis Funktionsfähiger oder erklärbarer Code. Teste das Ergebnis – korrekt oder fehlerhaft? Wie hilfreich ist die Erklärung des Modells?
⚙️ Hinweis: Gib immer den Kontext mit: Programmiersprache, Zweck, Eingabe/Ausgabe-Format. Ohne Kontext → generischer Code. Mit Kontext → gezielter Code.

Review & Abschluss

Was bleibt – und wie geht es weiter?

Kurze Ergebnisrunde: Was hat geklappt? Was hat überrascht? Wo lagen die Grenzen?

✅ Best Practices

  • Rolle + Aufgabe + Format angeben
  • Kontext immer mitliefern
  • Ergebnis kritisch prüfen
