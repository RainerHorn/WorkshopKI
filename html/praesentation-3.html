Tools & Plattformen

Womit laufen lokale LLMs?

🦙 Ollama
Kommandozeilen-Tool zum Laden und Ausführen von Modellen. Einfachste Installation, API inklusive.
Empfohlen: Einstieg & Server
🖥️ LM Studio
Grafische Oberfläche zum Herunterladen und Testen von Modellen. Ideal für Einsteiger ohne Terminal.
Empfohlen: Einsteiger
🌐 Open WebUI
Browser-basierte Chat-Oberfläche auf Ollama-Basis. Sieht aus wie ChatGPT – läuft komplett lokal.
Empfohlen: Unterricht
💡 Kombination: Ollama + Open WebUI – leicht zu installieren, vertraut wirkende Oberfläche für die Gruppe.

Hardware & Performance

Was brauche ich, was kann ich erwarten?

Modellgröße RAM-Bedarf Geschwindigkeit Qualität
3b (z. B. llama3.2:3b) ~4 GB Schnell (CPU) Grundlegend
7–8b (z. B. llama3.1:8b) ~8–10 GB Mittel (CPU/GPU) Gut
70b (z. B. llama3.3:70b) ~40+ GB Langsam ohne GPU Sehr gut
⚡ Empfehlung für Workshop-Laptops: llama3.2:3b (CPU reicht) oder mistral:7b mit 16 GB RAM. GPU (VRAM) beschleunigt massiv – ohne GPU ist CPU-Inferenz langsamer, aber funktionsfähig.

Sicherheit & Datenschutz

Warum lokale LLMs im Schulkontext sinnvoll sind

✅ Lokale Datenhaltung

  • Alle Daten bleiben auf dem Gerät
  • Kein Transfer in die Cloud
  • Kein Nutzerkonto erforderlich

🔒 DSGVO-Konformität

  • Keine Weitergabe an Dritte
  • Schülerdaten sicher verarbeitbar
  • Offline-Betrieb = kein Risiko

🌐 Netzwerk-Unabhängigkeit

  • Funktioniert ohne Internet
  • Kein Schul-Proxy-Problem
  • Stabile Performance

⚠️ Zu beachten

  • Modelle können falsche Infos liefern
  • Kein Filter für unangemessene Inhalte
  • System-Prompt zur Steuerung nutzen

Grenzen lokaler Modelle

Realistische Erwartungen setzen

Lokale LLMs sind leistungsfähig – aber kein Ersatz für große Cloud-Modelle. Diese Grenzen sollte man kennen:

📉 Qualitätsunterschied

  • Kleinere Modelle halluzinieren häufiger
  • Komplexe Schlussfolgerungen weniger präzise

💾 Ressourcenbedarf

  • RAM-Hunger: 8 GB+ für nutzbare Modelle
  • Ohne GPU: deutlich langsamer

📅 Aktualität

  • Wissen endet am Trainings-Cutoff
  • Keine aktuellen Ereignisse bekannt

🌍 Mehrsprachigkeit

  • Deutsch meist schlechter als Englisch
  • Englische Prompts oft bessere Ergebnisse
🎯 Fazit: Für strukturierte, klar definierte Aufgaben sind lokale LLMs sehr gut geeignet.

Was wir jetzt gemeinsam machen

1️⃣ Modell starten

Ollama starten, Modell laden:
ollama run llama3.2

2️⃣ Erster Prompt

Einfache Frage stellen – Antwort beobachten. Token-Ausgabe live sehen.

3️⃣ Schlechter Prompt

„Erkläre mir alles über KI." → vage, ausschweifend, wenig nützlich.

4️⃣ Guter Prompt

Rolle + Aufgabe + Format + Beispiel → präzise, nutzbare Antwort.

🔁 Reproduzierbarkeit: gleiche Anfrage → ähnliche Antwort. Temperature steuert Kreativität vs. Konsistenz.
